## <a name="features">🔋 Features</a>
👉 **Book Browsing**: View all available books  
👉 **Book Details**: Detailed view for each book  
👉 **Download**: Direct book download  
👉 **Responsive Design**: Mobile-first approach  
👉 **Fast Loading**: Built with Next.js

## <a name="tech-stack">⚙️ Tech Stack</a>
- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel Deployment

## <a name="quick-start">🤸 Quick Start</a>
```bash
git clone https://github.com/Abhias2405/eBook_Lib_Client
cd eBook_Lib_Client
npm install
npm run dev
```

## <a name="pages">🚀 Pages</a>
- `/` - Home page with book list
- `/books/[id]` - Individual book page
- Book download handler
